# 终端限速自动化测试用例

## 概述

本文档描述终端限速模块的自动化测试用例，包括IP限速和MAC限速两个子模块。测试步骤参照VLAN设置自动化测试的结构进行设计。

**测试设备**: 10.66.0.150
**账号**: admin
**密码**: admin123

## 一、IP限速测试用例

### 1.1 页面元素和功能

#### 列表页面字段
| 字段 | 说明 |
|------|------|
| 名称 | 限速规则名称 |
| 线路 | 关联的线路（可排序） |
| 内网地址 | IP地址或IP段 |
| 协议 | tcp/udp/tcp+udp/任意 |
| 限速模式 | 独立限速/共享限速（可排序） |
| 源端口 | 源端口号 |
| 目的端口 | 目的端口号 |
| 上行限速 | 上行带宽限制（可排序） |
| 下行限速 | 下行带宽限制（可排序） |
| 周期 | 生效时间周期 |
| 备注 | 备注信息 |
| 操作 | 编辑/删除 |

#### 表单字段
| 字段 | 类型 | 是否必填 | 说明 |
|------|------|----------|------|

### 1.2 IP限速测试步骤

#### 步骤1：验证IP限速页面
- 操作：登录设备，导航到 网络配置 > 终端限速 > IP限速
- 验证：
  - 页面标题显示"终端限速"
  - IP限速标签页处于选中状态
  - 页面正常加载，无错误信息

#### 步骤2：清理已有数据
- 操作：检查IP限速列表，如有多余数据则删除
  - 获取当前规则数量
  - 如果数量 > 0：
    - 点击表头全选复选框
    - 点击批量删除按钮
    - 确认删除
  - 刷新页面验证清理结果
  - **清理路由对象数据**：
    - 导航到 网络配置 > 路由对象 页面（URL: `#/networkConfiguration/routingObject`）
    - 点击"IP分组"标签页
    - 如有数据，全选并删除
    - 点击"时间计划"标签页
    - 如有数据，全选并删除
    - 返回IP限速页面
- 验证：
  - 列表为空，显示"暂无内容"
  - 确保测试环境干净
  - IP分组和时间计划也已清理

#### 步骤3：二次检查测试数据
- 操作：确保测试数据已清理
- 验证：无残留测试数据

#### 步骤4：创建测试用IP分组（在添加规则时自动创建）
- 操作：在添加IP限速规则时自动创建IP分组
  - 点击"添加"按钮打开新增限速弹窗
  - 点击"IP分组"下拉框
  - 在弹出的选择对话框中点击"创建分组"按钮
  - 选择"创建IPv4分组"
  - 分组名称：test_ip_group_001
  - IP列表：192.168.10.1-192.168.10.100
  - 点击"确定"保存分组
  - 选择刚创建的分组
- 验证：
  - IP分组创建成功
  - IP分组下拉框中可选择该分组

#### 步骤5：创建测试用时间计划
- 操作：如果没有可用的时间计划，需要先创建
  - 导航到 网络配置 > 路由对象 页面（URL: `#/networkConfiguration/routingObject`）
  - 点击"时间计划"标签页
  - 点击"添加"按钮
  - 计划名称：test_time_plan_001
  - 计划类型：按周循环
  - 生效日期：勾选周一至周五（09:00-18:00工作时间段）
  - 生效时间：09:00-18:00
  - 点击"保存"
  - 返回IP限速页面
- 验证：
  - 时间计划创建成功
  - 列表中显示新创建的时间计划
  - 返回IP限速页面继续测试

  - 关闭弹窗

## 五、注意事项

1. **测试前准备**：确保测试设备可访问，测试账号有效
2. **数据清理**：测试开始前清理已有数据，测试结束后清理测试数据
3. **IP分组/MAC组创建**：在添加限速规则弹窗内，通过下拉框的"创建分组"按钮创建，无需导航到其他页面
4. **时间计划创建**：在 网络配置 > 路由对象 > 时间计划 页面创建，可被多个限速规则引用
5. **数据隔离**：每个测试用例应独立，测试完成后清理数据
6. **异常处理**：关注系统对异常输入的处理方式
7. **边界条件**：测试限速值的边界值（0、最大值等）
8. **时间相关**：注意生效时间的各种组合情况
9. **参照VLAN测试**：步骤顺序、导入导出实现参照VLAN设置自动化测试
10. **路由对象导航**：URL为 `#/networkConfiguration/routingObject`（注意是 `routingObject` 不是 `routeObject`），IP分组和时间计划通过标签页切换
11. **已知前端bug**：限速值输入框允许非数字输入（如"abc"），保存时后端会转为0，测试代码需处理此情况
